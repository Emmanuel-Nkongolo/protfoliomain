Store dashboard

A modern store dashboard from which you can make all possible changes to your actual store, and they will be reflected in real time.

This project was built with TypeScript, Next.js, Clerk, Prisma, and a SQL database connected to Stripe for payments. I used technologies like Shadcn, Lucide-React, Axios, React-Hot-Toast, Zustand, and Cloudinary. You can check it out here.

the actual Store

A store built with Next.js, Shadcn, and React-Hot-Toast.
